# -Luxury-Housing-Data-Analysis
End-to-end data analytics project analyzing luxury housing data in Bangalore using Python, SQL, and Power BI. The project includes data cleaning, exploratory data analysis (EDA), SQL-based querying, and an interactive dashboard to uncover insights on pricing trends, buyer segments, and sales performance.

## Project Overview
This project analyzes luxury housing data in Bangalore to extract insights on pricing, buyer trends, and sales performance.

---
##  Tools Used
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI

---
##  Key Analysis
- Price distribution across locations
- Sales trend by quarter
- Buyer type analysis
- Revenue by sales channel
- Popular micro-markets

---
## Dashboard Insights
- High demand in premium locations
- Certain price ranges dominate transactions
- Online and broker channels drive most sales

---
## Project Structure
- data → cleaned dataset
- notebooks → data cleaning & EDA
- sql → SQL queries
- powerbi → dashboard file

---
##Conclusion
This project helps understand real estate market trends and supports data-driven decision making.
